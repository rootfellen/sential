
# Sential



<p align="center">
  <a href="https://sential.vercel.app/">
  <img src="/public/sential.png" width="256"/>
  </a>
  <p align="center">AI Saas Platform to generate code, images, audio & video by using OpenAI API & Replica API.</p>
</p>


## Features

- Stripe payment integration.

- Authentication.

- Monthly membership.

- Image, Video, Audio, Code generation

- Chat Assistant.

- Manage Account.

- Responsive UI

- Manage monthly subscription


## Tech stack

Front-end:

- Next.js

- Typescript

- Crisp

- Stripe

- Tailwind

- Shadcn UI

- Zod

- Zustand

Back-end:

- Prisma

- OpenAI & Replica API.

- Planetscale (MySQL)

- Vercel

- Axios

- Clerk
